# Оптическое распознавание рукописного текста (CV)

CNN-модель для распознавания рукописных символов и OCR-пайплайн для чтения сканированных документов.

## Задача

Распознать рукописный текст на зашумлённых сканах документов: предобработка изображений, сегментация символов, классификация, оценка качества распознавания.

## Стек

Python, PyTorch, OpenCV, Levenshtein

## Что сделано

- Предобработка зашумлённых сканов: медианная фильтрация шума и бинаризация для устойчивого поиска контуров символов
- Обучена CNN-модель классификации символов на датасете EMNIST (нелинейная активация, пулинг, дропаут)
- Реализованы метрики оценки качества: CER (Character Error Rate, расстояние Левенштейна) и WER (Word Error Rate)

## Результат

- **93.7% test accuracy** классификатора символов на EMNIST после 10 эпох обучения
- На полном end-to-end пайплайне (сегментация + распознавание реального скана) итоговое качество оказалось низким — узкое место определено как сегментация слитного/курсивного почерка: контурный подход плохо справляется со слипшимися буквами
- Направление для дальнейшей доработки: замена contour-based сегментации на более устойчивый подход (например, sliding window или отдельная модель сегментации)
