## 🛠 Подробное описание проектов

### 1. Avito Shield: Прототип асинхронного конвейера модерации
* **Задача:** Автоматическое обнаружение мошенничества, скрытых контактов на изображениях и фильтрация некачественного контента.
* **Реализация:** Каскадная (short-circuit) логика обработки запроса для экономии ресурсов. Извлечение текста со сложных фонов с помощью **EasyOCR**, валидация данных через `pydantic-settings`.
* **Инженерия:** Полное покрытие интеграционными тестами (`pytest`), автоматический CI-пайплайн в GitHub Actions, контейнеризация через Docker (базовый `slim`-образ с оптимизацией слоев).

### 2. Предсказание вероятности сделки (Kaggle Avito Demand Prediction)
* **Задача:** Расчет метрики `deal_probability` для многокомпонентных объявлений классифайда.
* **Реализация:** Извлечение эмбеддингов изображений глубокой сетью (ResNet18 + PCA для снижения размерности), текстовый инжиниринг признаков (TF-IDF + SVD), кросс-фичи, валидация через K-Fold target encoding без утечки данных (*data leakage*). На итоговом шаге — ансамблирование градиентного бустинга LightGBM.

### 3. Обучение RL-агента в симулированной среде
* **Задача:** Оптимизация траектории робота-пылесоса для сбора мусора в условиях ограниченной видимости.
* **Реализация:** Построение Q-Learning модели в средах `Discrete` и `MultiDiscrete`. Успешно разрешена проблема частичной наблюдаемости (POMDP) за счет расширения пространства представлений состояний (*state representation*), что позволило преодолеть плато сходимости.

### 4. Оптическое распознавание текста (CV/OCR)
* **Задача:** Сегментация и распознавание символов на зашумленных сканах документов.
* **Реализация:** Пайплайн предобработки (медианная фильтрация, бинаризация, поиск контуров OpenCV). Обучение кастомной CNN-архитектуры на датасете EMNIST с использованием регуляризации (Dropout). Проведен детальный *Error Analysis* с метриками CER и WER для локализации узких мест пайплайна.

### 5. Система машинного перевода (NLP Seq2Seq)
* **Задача:** Разработка базовой модели мультиязычного перевода текстов.
* **Реализация:** Архитектура Encoder-Decoder с механизмом Attention. Для ускорения инференса и экономии VRAM применен пайплайн со смешанной точностью (`torch.cuda.amp.GradScaler`). Проведена серия контролируемых R&D-экспериментов со скоростью обучения и шедулерами для стабилизации функции потерь.

## 📐 Фундаментальная математическая база
Все проекты опираются на академическую подготовку в рамках специализации «Искусственный интеллект»:
* **Линейная алгебра:** Матричные разложения (SVD, QR), метод наименьших квадратов, проекции.
* **Математический анализ:** Градиентные методы оптимизации, многомерный анализ, функции потерь.
* **Теория вероятностей и мат. статистика:** Проверка гипотез, статистическое моделирование, параметрические распределения.
