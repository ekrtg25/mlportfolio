# Умный робот-пылесос (RL)

Q-Learning агент, обучающийся убирать комнату в симулированной среде Gymnasium.

## Задача

Обучить агента с дискретным пространством состояний и действий убирать всю грязь в комнате за минимальное число шагов, используя Q-Learning и Reward Engineering.

## Стек

Python, Gymnasium, NumPy

## Что сделано

- Спроектирована среда на базе Gymnasium с дискретным пространством состояний/действий (Discrete, MultiDiscrete)
- Реализован Q-Learning агент с эпсилон-жадной стратегией и затуханием epsilon
- Инжиниринг функции вознаграждения через `CustomRewardWrapper`
- В процессе итераций обнаружена и устранена проблема с урезанным представлением состояния (state), из-за которой агент не мог обучиться нужному поведению — после исправления обучение сошлось

## Результат

**100% успешных завершений уборки на 200 тестовых эпизодах, в среднем 15 шагов до полной уборки комнаты.**

Путь к результату не был прямым: первые две версии агента (с разными reward-функциями) давали 0% успеха. Финальная версия сработала после исправления бага в представлении состояния — история диагностики отражена в самом ноутбуке.
