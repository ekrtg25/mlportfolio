# Предсказание вероятности сделки по объявлению — Kaggle: Avito Demand Prediction Challenge

Регрессионная модель на реальных данных объявлений Avito: предсказание `deal_probability` (вероятность, что объявление приведёт к сделке).

## Задача

[Avito Demand Prediction Challenge](https://www.kaggle.com/competitions/avito-demand-prediction) — предсказать вероятность сделки по объявлению на основе текста, категорий, цены, геоданных и изображения. Метрика соревнования — RMSE.

## Стек

Python, LightGBM, Pandas, Scikit-learn, PyTorch (ResNet18 для эмбеддингов изображений)

## Что сделано

Пайплайн строился итеративно, с последовательным наращиванием фичей и контролем метрики на валидации на каждом шаге:

1. **Базовые фичи**: цена, категориальные признаки (регион, город, категория, тип пользователя), дата размещения
2. **Текстовые фичи**: TF-IDF + TruncatedSVD по `title` и `description`, длины текстов, счётчики знаков препинания/заглавных букв
3. **Агрегации по цене**: среднее и std цены по категориям и регионам, отклонение цены объявления от среднего
4. **Эмбеддинги изображений**: признаки из предобученной ResNet18 (без последнего слоя), сжатые через IncrementalPCA
5. **Target Encoding**: K-Fold энкодинг категориальных признаков без утечки данных (leakage-free)
6. **Кросс-фичи**: произведения текстовых и визуальных компонент (text × image), Z-score цены внутри города/категории

Модель — **LightGBM** (регрессия, метрика RMSE), обучение с early stopping и валидацией на отложенной выборке.

## Результат

**RMSE = 0.23117** на публичном лидерборде Kaggle (для сравнения: топовые решения на момент проведения конкурса — около 0.214–0.218).
