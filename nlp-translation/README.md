# Система машинного перевода (NLP) — Seq2Seq + Attention

Перевод текстов в молодёжный стиль на базе архитектуры Encoder-Decoder с механизмом Attention.

## Задача

Обучить нейросетевую модель перевода (Seq2Seq) на собственном датасете пар текстов, оценить качество с помощью стандартных метрик машинного перевода.

## Стек

Python, PyTorch, NLTK, Sacrebleu

## Что сделано

- Проведён EDA датасета: анализ длин текстов, проверка пропусков и дубликатов
- Реализована архитектура Encoder-Decoder (GRU) с механизмом Attention
- Метрики BLEU и chrF реализованы вручную (с нуля) и через `sacrebleu`, покрыты unit-тестами
- Пайплайн обучения: оптимизатор Adam, `ReduceLROnPlateau`, Mixed Precision (`torch.cuda.amp.GradScaler`) для ускорения на GPU

## Результат и честная рефлексия

В ходе обучения модель не смогла превзойти тривиальный бейзлайн (копирование исходного текста без перевода): финальный чекпоинт давал вырожденный вывод и BLEU/chrF ниже бейзлайна. По логам это похоже на коллапс модели (model collapse) — вероятно, связано с балансом learning rate/teacher forcing или маскированием PAD-токенов в лоссе; несколько попыток перебора гиперпараметров и шедулеров не привели к сходимости на валидации.

Это открытый пункт для доработки: причина нестабильности обучения диагностирована лишь частично, точка входа для дальнейшего дебага — сравнение loss по батчам и проверка teacher forcing ratio.
