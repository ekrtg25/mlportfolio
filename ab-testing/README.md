# Статистическое моделирование A/B и оценки кандидатов

Модель для определения оптимального порога числа вопросов при отборе кандидатов на основе биномиального распределения.

## Задача

Определить, сколько вопросов нужно задать кандидату, чтобы одновременно максимизировать вероятность прохождения сильных кандидатов и отсеять слабых.

## Стек

Python, SciPy (`scipy.stats.binom`), Pandas

## Что сделано

- Анализ биномиального распределения вероятностей правильных ответов для двух групп кандидатов (сильные/слабые)
- Определено оптимальное количество вопросов, при котором вероятность прохождения сильных кандидатов ≥ 80%, а отсева слабых ≥ 70%
